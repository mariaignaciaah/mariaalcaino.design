Modelado Superficies / Fusion 360

Modelado mediante superficies en FUSION 360.

TUTORIAL

  • En el módulo de Surface: muy útil para hacer envases o contenedores, porque parte de una geometría específica.
  1. Spline en Sketch para dibujar la geometría. Construct para hacer plano de construcción sobre la geometría el sketch.
  2. Loft en surface, siempre que yo modelo 2 superficies puedo generar la intersección exacta entre ellas. (loft en solid, se crea como un body)
  3. Las superficies no tienen espesor, solo son la continuidad de una superficie, una superficie que intersecta otra superficie.
  4. Patch para tapar las superficies.
  5. Necesitamos llegar a un sólido para poder fabricar el elemento. Para eso vamos a Stich, así todas las superficies se transforman en un solo body o sólido. Queda como un sólido completo, y lo que nosotros queremos es que sea un contenedor.
  6. Para eso vamos al módulo de sólido y hacemos un Fillet, y luego usamos el comando Shell para hacer un vaceado, transforma un sólido en una caja.
