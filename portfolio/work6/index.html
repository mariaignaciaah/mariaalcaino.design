BITÁCORA INSTALACIÓN HUGO

COMO INSTALAR HUGO EN MAC

  • Primero debes crear una cuenta en GitHub. Además debes descargar Hugo, Visual Studio y Homebrew.
  • Luego debes abrir el Terminal e instalar Homebrew dentro de éste. Durante este proceso te pediran la clave del equipo.
  • Seguido de esto, debes escribir los siguientes códigos dentro del terminal.
  1. cd documents (para ir a la carpeta de documentos)
  2. cd hugo (para ir a la carpeta hugo)
  3. hugo new site ….. (crear un nuevo sitio, en los ….. debe ir el nombre del sitio )
  • Luego de ingresar los códigos mencionados en el terminal, debes descargar un tema de la página themes.gohugo.io/, y seguir los siguientes pasos.
  1. Descomprimir la carpeta del tema descargado.
  2. Eliminar del nombre de la carpeta “-master”.
  3. Copiar y pegar la carpeta del tema en la carpeta THEMES que esta en el interior de la carpeta de la página web.
  4. Luego de la carpeta del tema debes abrir la carpeta EXAMPLESITE y cortar y pegar todos los archivos en el interior de la carpeta del tema descargado.
  • Luego de seguir estos pasos, debes volver al terminal e ingresar el codigo:
  1. cd documents (la carperta donde se encuentra HUGO)
  2. cd hugo
  3. cd website (website es en mi caso el nombre de mi carpeta del sitio)
  4. hugo server
  • Luego de dar el hugo server, aparecera una lista de especificaciones del sitio creado (que es un local host), el cual no esta conectado a la red.

  • En la parte de abajo se encuentra el link para abrir en el servidor.

http://localhost:1313/
